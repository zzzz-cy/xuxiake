# 徐霞客游记可视化

### 项目配置
git clone本项目到本地后，进入xuxiake文件夹，执行以下命令：
```
npm install
```

### 项目试运行
在项目文件夹中执行以下命令：
```
npm run serve
```

### 项目编译
```
npm run build
```

### 内部修改教学
1.git clone项目到本地

2.完成修改

3.在项目文件夹中选择 Git Bash Here

4.按顺序执行以下命令

```
git add *
git commit  -m  "对你更新文件的简单注释，比如：修改字体"
git push
```
